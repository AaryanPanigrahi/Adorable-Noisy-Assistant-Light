# Adorable-Noisy-Assistant-Light (Ongoing)

A robotic companion combining the charm of the traditional Chinese lucky cat (Maneki-neko) with ring light on its head for making content instead

## Project Overview

Created as a thoughtful gift for someone who loves making videos and cats

- **Ring Light Control:** Adjustable brightness and hue settings
- **Neck Rotation:** Servo-controlled head movement
- **I/O:** Back-mounted ports for programming and power connections

### Hardware

- Arduino Uno
- Servo motor
- Programmable LED ring light (Adafruit)
- Custom 3D-printed cat chassis
- Rechargable LiPo battery
